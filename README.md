# Password Generator

Generate strong and secure passwords with Python for your accounts.

## Overview

This Python script allows you to generate strong and secure passwords with various customization options. You can specify the length of the password and choose whether to include uppercase letters, lowercase letters, digits, and special characters.

## Features

- Customizable password length.
- Options to include/exclude uppercase letters, lowercase letters, digits, and special characters.
- Command-line interface for easy usage.

## Usage

### Prerequisites

Make sure you have Python installed on your machine.

### Running the Script

1. Clone the repository:
   
   ```bash
   git clone https://github.com/atharv-451/Password-Generator.git
   
2. Navigate to the Respository:
   
   ```bash
   cd password-generator
   
3. Run the Python Script:

   python password_gen.py
