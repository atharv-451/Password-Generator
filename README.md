# Password Generator

Generate strong and secure passwords with Python for your accounts.

## Overview

This Python script generates strong passwords with a combination of uppercase letters, lowercase letters, digits, and special characters.

## Features

- Simple and easy-to-understand password generation.
- Customizable password length.

## Usage

### Prerequisites

Make sure you have Python installed on your machine.

### Running the Script

1. Clone the repository:
   
   ```bash
   git clone https://github.com/atharv-451/Password-Generator.git
   
2. Navigate to the Respository:
   
   ```bash
   cd password-generator
   
3. Run the Python Script:

   ```bash
   python password_gen.py
